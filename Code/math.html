Algebraic Expression & Identities, Factorization



Algebraic Expression is a Combination of integers, variables and operations. Ex - 2(x + 8y). The main point of these expression is that they don't equate to anything. Now though you can't solve expression because it does'nt equate to anything but still you can very much simplify them. One such way is by Highest Common Factior or HCF.

Remove the highest common factor from an algebraic statement and position it in front of the brackets to factorise it. After that, divide each term by the highest common factor to get the phrase inside the brackets.
Ex- 7x+35 => 7x=7*x => 35=7*5 Common factor=7 Thus, the simplified equation is 7(x+5).

Now the 2nd way is using Algebraic Identities:

There are multiple identities which we can use to simplify them but I will show only three Identities and also show an example.
=>(a + b)2 = a2 + 2ab +b2, =>(a - b)2 = a2 - 2ab + b2 =>a2 – b2 = (a + b)(a – b)
Now these are a few of those identities which are used to calculate Algebraic Expressions.
Ex-z2 + 62 + 9
We can express z2 + 6z + 9 as using a2 + 2ab + b2 = (a + b)2 = (z)2 + 2(z)(3) + (3)2
= (z + 3)2
= (z + 3)(z + 3) (Source: math-only-math.com)

Note: This is applicable only on Trinomials.

This method is called the Sum and Product Factorizing. This method is used to solve Quadratic Equations. This method is based on this expression (ax2+bx+c) with the value of a=1. This method is also used the most.

Ex-x2+7x+12

How to calculate the root of a Linear Equation? Simple way of finding the root of a linear equation with one variable is to ship all the constants to the RHS or Right-Hand-Side, then since the sides are changing the operations will change as well and then perform the operation and you find the root of the equation which is the value of the variable.

Ex- x+4=7.

=>x = 7-4 =>x = 3

This is also a solution to solve a formed linear equation. Now the question arises how do you form a linear equation? Well make sure that the equation you form is in the slope-intercept form of y = mx+b in which x and y are variable.

Arithmetic mean of a grouped data


The first step in calculating the mean of grouped data is to determine the midpoint of each interval or class. The frequencies of the associated classes must then be multiplied by these midpoints. The value of the mean is the sum of the products divided by the total number of values.
